# 40-satang-coding68
ฝึกเขียนโปรแกรม
1 ชื่อโปรแกรม การปิดไฟอัตโนมัติ
สามารถบอกการปิดไฟได้

2 ความสารถของโปรแกรม
ฟีเจอร์ที่ 1 : ตรวจจับไฟอัตโนมัติ
ฟีเจอร์ที่ 2 : มีเสียงแจ้งเตือนว่าลืมปิดไฟ
ฟีเจอร์ที่ 3 : สั่งปิดไฟอัตโนมัติ
ฟีเจอร์ที่ 4 : ควบคุมผ่านแอป

3 เคื่องมือที่ใช้ 
Language : Python
Libary/Framework: Tkinter, CustomTkinter
Design Tool : Canva
